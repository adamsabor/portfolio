Objectif

L'objectif de ce projet est de créer un script Python permettant de générer le contenu d'un journal en fonction d'une date et d'une édition spécifique. Le projet repose sur l'utilisation de la programmation orientée objet pour structurer les articles et interviews, en évitant la duplication de code grâce à l'héritage.
